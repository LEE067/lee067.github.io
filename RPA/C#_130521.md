## 소수 만들기(Java)

```java
class Solution {
    
    public int solution(int[] nums) {
        int answer = 0;
        int len = nums.length;
        
        for (int i = 0; i < len - 2; i++){
            for (int j = (i + 1); j < len - 1; j++) {
                for (int k = (j + 1); k < len; k++) {
                    if (checkingPrime(nums[i] + nums[j] + nums[k]))
                        answer++;
                }
            }
        }
        return answer;
    }
    public boolean checkingPrime(int num) {
        for (int i = 2; i < num; i++) {
            if (num % i == 0) {
                return false;
            }
        }
        return true;
    }
}
```
